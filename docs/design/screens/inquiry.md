# 引合

## 引合一覧

### 要素

| 要素 | 内容 |
|---|---|
| フィルタ | ステータス（new / converted / declined）、経路（inquiry_source の各値） |
| 検索 | 顧客名・件名で部分一致検索 |
| ソート | 登録日（デフォルト: 新しい順） |
| 一覧テーブル | 件名、顧客名、経路、ステータス、登録日 |
| 新規作成ボタン | 引合登録画面への遷移 |

### 状態

| 条件 | 表示 |
|---|---|
| 結果 0 件 | 「該当する引合はありません」 |
| ステータスが converted | ステータス列に案件へのリンクを併記 |

### 遷移

| 操作 | 遷移先 |
|---|---|
| 行をクリック | 引合詳細 |
| 新規作成ボタン | 引合登録 |

## 引合詳細

### 要素

| 要素 | 内容 |
|---|---|
| 基本情報 | 件名、問い合わせ内容（Markdown プレビュー）、概要（Markdown プレビュー）、経路、ステータス、登録日。編集可能 |
| 顧客セクション | 紐づいている顧客名と詳細へのリンク。未設定の場合は顧客選択/新規作成コントロール |
| 商談記録リスト | この引合に紐づく商談の一覧（日時、種別、要旨）。新しい順 |
| 商談追加ボタン | 商談登録への遷移（inquiryId 自動設定） |
| ステータス操作 | 現在のステータスに応じたアクションボタン |
| 案件リンク | ステータスが converted の場合、生成された案件へのリンク |

### 状態

| 条件 | 表示 |
|---|---|
| status = new | 「案件化する」「見送る」ボタンを表示 |
| status = new かつ clientId = null | 「案件化する」ボタンを無効化し「顧客の紐づけが必要です」と表示 |
| status = new かつ案件化に承認が必要 | 「案件化する」を押すと承認リクエスト生成。「承認リクエストを作成しました」のフィードバック |
| status = new かつ承認待ち | 「承認待ちです」のバナーを表示。承認リクエスト詳細へのリンク |
| status = converted | 「案件化済み」のラベルと案件へのリンク。ステータス操作ボタンなし |
| status = declined | 「見送り済み」のラベル。「見送りを撤回する」ボタンを表示 |

### 遷移

| 操作 | 遷移先 |
|---|---|
| 顧客名クリック | 顧客詳細 |
| 商談をクリック | 商談詳細 |
| 商談追加ボタン | 商談登録（inquiryId 設定済み） |
| 案件リンク | 案件詳細 |
| 案件化成功（承認不要） | 案件詳細（新規作成された案件） |

### 操作

| 操作 | 権限 | 結果 |
|---|---|---|
| 基本情報を編集 | admin / manager / member | 引合情報が更新される |
| 顧客を紐づけ | admin / manager / member | clientId が設定される |
| 案件化する | admin / manager | 承認要否を判定し、案件生成または承認リクエスト生成 |
| 見送る | admin / manager | ステータスが declined になる |
| 見送りを撤回 | admin / manager | ステータスが new に戻る |
| 削除 | admin | 引合が削除される（確認ダイアログ） |

## 引合登録

### 要素

| 要素 | 内容 |
|---|---|
| 顧客 | 顧客選択コントロール。任意。「新規作成」でインライン顧客登録 |
| 件名 | テキスト入力。必須 |
| 経路 | セレクト（inquiry_source の各値）。必須 |
| 社内担当者 | ユーザー選択。任意 |
| 問い合わせ内容 | テキストエリア。任意。placeholder: 「メール原文、電話メモなど」 |
| 概要 | テキストエリア。任意 |
| 登録ボタン | |

### 遷移

| 操作 | 遷移先 |
|---|---|
| 登録成功 | 引合詳細（作成された引合） |
| キャンセル | 引合一覧 |
