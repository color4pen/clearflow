# 顧客

## 顧客一覧

### 要素

| 要素 | 内容 |
|---|---|
| 検索 | 企業名で部分一致検索 |
| フィルタ | 業種 |
| 一覧テーブル | 企業名、業種、関連案件数、登録日 |
| 新規作成ボタン | |

### 遷移

| 操作 | 遷移先 |
|---|---|
| 行をクリック | 顧客詳細 |

---

## 顧客詳細

### 要素

| 要素 | 内容 |
|---|---|
| 企業情報 | 企業名、業種、規模、所在地、備考。編集可能 |
| 担当者リスト | 担当者一覧（氏名、部署、役職、メール、電話、主担当フラグ）。追加・編集可能 |
| 関連引合リスト | この顧客に紐づく引合の一覧（件名、ステータス、登録日） |
| 関連案件リスト | この顧客に紐づく案件の一覧（案件名、フェーズ、想定金額） |

### 遷移

| 操作 | 遷移先 |
|---|---|
| 引合をクリック | 引合詳細 |
| 案件をクリック | 案件詳細 |

---

## 顧客登録

### 要素

| 要素 | 内容 |
|---|---|
| 企業名 | テキスト入力。必須 |
| 業種 | テキスト入力。任意 |
| 規模 | テキスト入力。任意 |
| 所在地 | テキスト入力。任意 |
| 備考 | テキストエリア。任意 |

### 遷移

| 操作 | 遷移先 |
|---|---|
| 登録成功 | 顧客詳細（作成された顧客） |
| キャンセル | 顧客一覧 |
