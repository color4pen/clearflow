# 案件

## 案件一覧

### 要素

| 要素 | 内容 |
|---|---|
| フィルタ | フェーズ、顧客、契約形態のドロップダウン |
| フェーズサマリ | フェーズごとの件数と想定金額合計 |
| 一覧 | 案件名、顧客名、フェーズ、契約形態、想定金額 |
| 新規作成ボタン | 案件登録画面への遷移 |

### 遷移

| 操作 | 遷移先 |
|---|---|
| 行をクリック | 案件詳細 |
| 新規作成ボタン | 案件登録 |

## 案件詳細

### 要素

| 要素 | 内容 |
|---|---|
| 基本情報 | 案件名、概要、フェーズ、契約形態、想定金額、想定開始日、想定終了日。編集可能 |
| フェーズ変更 | 現在のフェーズの表示とフェーズ変更コントロール |
| 顧客セクション | 顧客名と詳細へのリンク |
| 担当者セクション | DealContact の一覧（担当者名、役割）。追加・削除操作 |
| 商談記録リスト | この案件に紐づく商談 + 元の引合に紐づく商談。種別と日時を表示。新しい順 |
| 商談追加ボタン | 商談登録への遷移（dealId 自動設定） |
| アクションアイテムセクション | action_items テーブルから案件に紐づくアクションアイテムを取得して表示。カード形式（1 行目に説明、下段に担当者・期日を縦積み）。チェックボックスで完了切替。「追加」ボタンでモーダル作成。行クリックで編集モーダル（削除はモーダル左下） |
| 契約セクション | 紐づく契約の一覧（契約名、金額、ステータス）。受注後のみ表示 |
| 契約作成ボタン | 契約登録への遷移（dealId 自動設定）。受注後のみ表示 |
| 備考セクション | Markdown 対応テキストエリア。案件のメモ・共有事項 |
| 見積承認リンク | estimateRequestId がある場合、承認リクエストへのリンク |
| 引合リンク | inquiryId がある場合、元の引合へのリンク |

### 状態

| 条件 | 表示 |
|---|---|
| phase が非終端 | フェーズ変更コントロールを表示。全非終端フェーズ + 受注 + 失注を選択肢として表示 |
| phase = won | 「受注」ラベル。フェーズ変更不可。契約セクションと契約作成ボタンを表示 |
| phase = lost | 「失注」ラベル。フェーズ変更不可。契約セクション非表示 |
| 受注/失注を選択時 | 確認ダイアログ「この操作は取り消せません。よろしいですか？」 |
| 担当者 0 件 | 「担当者を追加してください」 |

### 遷移

| 操作 | 遷移先 |
|---|---|
| 顧客名クリック | 顧客詳細 |
| 担当者名クリック | 顧客詳細（担当者セクション） |
| 商談をクリック | 商談詳細 |
| 商談追加ボタン | 商談登録（dealId 設定済み） |
| アクションアイテムの行をクリック | 編集モーダル（内容・担当者・期日の変更、削除） |
| 契約をクリック | 契約詳細 |
| 契約作成ボタン | 契約登録（dealId 設定済み） |
| 引合リンク | 引合詳細 |

### 操作

| 操作 | 権限 | 結果 |
|---|---|---|
| 基本情報を編集 | admin / manager / member | 案件情報が更新される |
| フェーズ変更（非終端） | admin / manager / member | フェーズが変更される |
| 受注 / 失注 | admin / manager | フェーズが won / lost に遷移（不可逆） |
| 担当者を追加 | 全ロール | DealContact が作成される |
| 担当者を削除 | 全ロール | DealContact が削除される |
| アクションアイテムを追加 | 全ロール | モーダルで作成。action_items テーブルにレコードが作成される |
| アクションアイテムを編集 | 全ロール | モーダルで内容・担当者・期日を更新 |
| アクションアイテムを完了/未完了 | 全ロール | done が反転する |
| アクションアイテムを削除 | admin | 編集モーダル左下から削除（確認ダイアログ） |
| 案件を削除 | admin | 確認ダイアログ後に削除される。商談・契約が紐づく場合は削除不可 |

## 案件登録

### 要素

| 要素 | 内容 |
|---|---|
| 顧客 | 顧客選択。必須。「新規作成」でインライン顧客登録 |
| 案件名 | テキスト入力。必須 |
| 契約形態 | セレクト（contract_type）。任意 |
| 想定金額 | 数値入力。任意 |
| 想定開始日 | 日付入力。任意 |
| 想定終了日 | 日付入力。任意 |
| 営業担当 | ユーザー選択。任意 |
| 技術担当 | ユーザー選択。任意 |
| 備考 | テキストエリア。任意 |

### 遷移

| 操作 | 遷移先 |
|---|---|
| 登録成功 | 案件詳細（作成された案件） |
| キャンセル | 案件一覧 |
