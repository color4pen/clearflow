# 設定

## 承認ポリシー設定

### 要素

| 要素 | 内容 |
|---|---|
| ポリシー一覧テーブル | ポリシー名、トリガーアクション、条件、テンプレート名、有効/無効 |
| 新規作成ボタン | |
| 有効/無効トグル | 行ごとのトグル |

### ポリシー作成・編集フォーム

| 要素 | 内容 |
|---|---|
| ポリシー名 | テキスト入力。必須 |
| トリガーアクション | セレクト（trigger_action の各値）。必須 |
| 条件フィールド | テキスト入力。任意（未設定 = 常に承認を要求） |
| 条件演算子 | セレクト（comparison_operator）。条件フィールドが設定されている場合は必須 |
| 条件値 | テキスト入力。条件フィールドが設定されている場合は必須 |
| テンプレート | セレクト（既存テンプレートから選択）。必須 |

---

## 承認テンプレート設定

### 要素

| 要素 | 内容 |
|---|---|
| テンプレート一覧テーブル | テンプレート名、ステップ数、作成日 |
| 新規作成ボタン | |

### テンプレート作成・編集フォーム

| 要素 | 内容 |
|---|---|
| テンプレート名 | テキスト入力。必須 |
| 説明 | テキストエリア。任意 |
| ステップ定義リスト | 順序付きリスト。各ステップ: ステップ名、承認者ロール or 承認者ユーザー、期限日数 |
| ステップ追加・削除・並べ替え | |
| フォームフィールド定義リスト | 各フィールド: キー、ラベル、型、必須フラグ、選択肢 |
| フィールド追加・削除 | |

---

## 承認委任設定

### 要素

| 要素 | 内容 |
|---|---|
| 自分の委任一覧 | 委任先、期間、有効/無効 |
| 新規作成ボタン | |
| 全委任一覧 | admin のみ。全ユーザーの委任 |

### 委任作成フォーム

| 要素 | 内容 |
|---|---|
| 委任先ユーザー | ユーザー選択。必須 |
| 開始日 | 日付入力。必須 |
| 終了日 | 日付入力。必須 |

---

## ユーザー管理

### 要素

| 要素 | 内容 |
|---|---|
| ユーザー一覧テーブル | ユーザー名、メールアドレス、ロール |
| ロール変更 | 行ごとのセレクト（admin のみ操作可能） |

---

## Webhook 設定

### 要素

| 要素 | 内容 |
|---|---|
| エンドポイント一覧 | URL、購読イベント数、有効/無効、直近の配信状態 |
| 新規作成ボタン | |

### エンドポイント作成・編集フォーム

| 要素 | 内容 |
|---|---|
| URL | テキスト入力。必須 |
| シークレット | テキスト入力。必須 |
| 購読イベント | チェックボックスリスト（イベント種別の一覧） |

### 配信ログ

| 要素 | 内容 |
|---|---|
| 配信一覧 | イベント種別、配信日時、ステータス、レスポンスコード |

---

## 監査ログ

### 要素

| 要素 | 内容 |
|---|---|
| フィルタ | 操作者、操作種別、対象種別、期間 |
| ログ一覧テーブル | 日時、操作者、操作内容、対象種別、対象名 |
| エクスポートボタン | CSV ダウンロード |

### 遷移

| 操作 | 遷移先 |
|---|---|
| 対象名をクリック | 対象エンティティの詳細画面 |
