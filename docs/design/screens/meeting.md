# 商談記録

## 商談詳細

### 要素

| 要素 | 内容 |
|---|---|
| 基本情報 | 日時、種別、議事要旨。編集可能 |
| 議事録 | Markdown 対応のテキストエリア。編集モードとプレビューモードの切替 |
| ヒアリング情報 | type = hearing の場合のみ表示。課題、予算、意思決定者、時期、競合の各フィールド |
| 出席者リスト | 社内ユーザーと外部参加者の一覧 |
| 出席者追加 | 社内ユーザー選択 / 顧客担当者選択 / 外部参加者の手動入力 |
| 顧客担当者登録チェックボックス | 外部参加者をクライアントの担当者として登録するオプション |
| アクションアイテムリスト | action_items テーブルからこの商談に紐づくアクションアイテムを取得して表示。チェックボックスで完了切替。行の「編集」ボタンでモーダル編集（削除はモーダル左下） |
| アクションアイテム追加 | 「追加」ボタンでモーダル作成。内容、担当者、期日を入力。meeting_id と deal_id が自動設定される |
| 紐づけ先リンク | 案件名 / 引合名へのリンク |

### 状態

| 条件 | 表示 |
|---|---|
| type = hearing | ヒアリング情報セクションを表示 |
| type ≠ hearing | ヒアリング情報セクションを非表示 |

### 操作

| 操作 | 権限 | 結果 |
|---|---|---|
| 基本情報・議事録を編集 | admin / manager / member | 商談が更新される |
| 出席者を追加・削除 | admin / manager / member | attendees が更新される |
| アクションアイテムを追加 | admin / manager / member | action_items テーブルにレコードが作成される |
| アクションアイテムを完了/未完了 | admin / manager / member | done が反転する |
| アクションアイテムを編集 | admin / manager / member | 内容・担当者・期日が更新される |
| アクションアイテムを削除 | admin | 確認ダイアログ後に削除される |
| 外部参加者を顧客担当者に登録 | admin / manager / member | ClientContact が作成される |

## 商談登録

### 要素

| 要素 | 内容 |
|---|---|
| 日時 | 日時入力。必須 |
| 種別 | セレクト（meeting_type）。必須 |
| 議事要旨 | テキスト入力。任意 |

**注**: 紐づけ先（dealId / inquiryId）は遷移元から自動設定される。

### 遷移

| 操作 | 遷移先 |
|---|---|
| 登録成功 | 商談詳細（作成された商談） |
| キャンセル | 遷移元（案件詳細 or 引合詳細） |
