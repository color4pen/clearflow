# タスク

## タスク一覧

### 要素

| 要素 | 内容 |
|---|---|
| フィルタタブ | 未完了 / 完了 |
| 担当者切替 | 自分のタスク（デフォルト）/ 全員 |
| 新規作成ボタン | モーダルでタスクを作成 |
| テーブル | チェックボックス、内容、担当者、期日、紐づけ先 |

### テーブルカラム

| カラム | 幅 | 内容 |
|---|---|---|
| チェック | 24px | 完了/未完了の切替（チェックボックス 18px） |
| 内容 | 1fr | タスクの説明 |
| 担当者 | 100px | ユーザー名 |
| 期日 | 100px | 日付（mono フォント） |
| 紐づけ先 | 140px | 案件名 / 商談（日付＋種別）/ 引合名 / —。別タブで開くリンク付き |

行クリックで編集モーダルを開く（独立した編集ボタンは持たない。チェックボックスと紐づけ先リンクは行クリックと独立して動作する）。

### 紐づけ先ピッカー

新規作成・編集とも、紐づけ先は「紐づけ先を選択」モーダル（ピッカー）で 1 つだけ選ぶ。

| 要素 | 内容 |
|---|---|
| タブ | 案件 / 引合 / 商談 の 3 タブ |
| 検索 | 各タブでサーバー検索（案件・引合はタイトル、商談は summary）。入力をデバウンスして取得 |
| 候補テーブル | ヘッダー常時表示。案件/引合は「顧客名・タイトル」、商談は「親（案件/引合）・日時/種別」。各行に対象画面を別タブで開く「開く」リンク |
| 選択 | 行クリックで 1 件選択。案件/引合/商談のいずれか単一に紐づく（選び直すと差し替え）。「なし」で紐づけを外す |

### 新規作成モーダル

| フィールド | 型 | 必須 | 説明 |
|---|---|---|---|
| 内容 | text | Yes | タスクの説明 |
| 担当者 | select | No | デフォルト: 自分 |
| 期日 | date | No | |
| 紐づけ先 | ピッカー | No | 案件 / 引合 / 商談 から 1 つ |

### 編集モーダル

新規作成と同じフィールド（内容、担当者、期日、紐づけ先）。行クリックで開く。左下に「削除」リンク（確認ダイアログ付き）。

### 状態

| 条件 | 表示 |
|---|---|
| 結果 0 件 | 「タスクはありません」 |
| デフォルト | 自分の未完了タスクのみ表示 |
| 完了済みタスク | 内容に取り消し線、テキスト色をミュート |

### 遷移

| 操作 | 遷移先 |
|---|---|
| 紐づけ先リンク（「開く」）クリック | 案件詳細 / 商談詳細 / 引合詳細（別タブで開く） |

### 操作

| 操作 | 権限 | 結果 |
|---|---|---|
| 新規作成 | 全ロール | アクションアイテムが作成される |
| 完了切替 | 全ロール | done が反転する |
| 編集 | 全ロール | 行クリックで編集モーダルを開き、内容・担当者・期日・紐づけ先が更新される |
| 削除 | admin | アクションアイテムが削除される（確認ダイアログ） |
