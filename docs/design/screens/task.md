# タスク

## タスク一覧

### 要素

| 要素 | 内容 |
|---|---|
| フィルタタブ | 未完了 / 完了 |
| 担当者切替 | 自分のタスク（デフォルト）/ 全員 |
| 新規作成ボタン | モーダルでタスクを作成 |
| テーブル | チェックボックス、内容、担当者、期日、紐づけ先、編集ボタン |

### テーブルカラム

| カラム | 幅 | 内容 |
|---|---|---|
| チェック | 24px | 完了/未完了の切替 |
| 内容 | 1fr | タスクの説明 |
| 担当者 | 100px | ユーザー名 |
| 期日 | 100px | 日付（mono フォント） |
| 紐づけ先 | 140px | 案件名 / 商談日 / 引合名 / —（リンク付き） |
| 操作 | 50px | 編集ボタン |

### 新規作成モーダル

| フィールド | 型 | 必須 | 説明 |
|---|---|---|---|
| 内容 | text | Yes | タスクの説明 |
| 担当者 | select | No | デフォルト: 自分 |
| 期日 | date | No | |
| 案件 | select | No | 紐づけ先の案件 |
| 引合 | select | No | 紐づけ先の引合 |

### 編集モーダル

新規作成と同じフィールド（内容、担当者、期日）。左下に「削除」リンク（確認ダイアログ付き）。

### 状態

| 条件 | 表示 |
|---|---|
| 結果 0 件 | 「タスクはありません」 |
| デフォルト | 自分の未完了タスクのみ表示 |
| 完了済みタスク | 内容に取り消し線、テキスト色をミュート |

### 遷移

| 操作 | 遷移先 |
|---|---|
| 紐づけ先クリック | 案件詳細 / 商談詳細 / 引合詳細 |

### 操作

| 操作 | 権限 | 結果 |
|---|---|---|
| 新規作成 | 全ロール | アクションアイテムが作成される |
| 完了切替 | 全ロール | done が反転する |
| 編集 | 全ロール | 内容・担当者・期日が更新される |
| 削除 | admin | アクションアイテムが削除される（確認ダイアログ） |
