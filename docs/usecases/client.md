# client

### 顧客を登録する

- **目的**: 取引先の企業情報を記録する
- **前提**: なし
- **権限**: 全ロール
- **結果**: 顧客が作成される
- **関連**: なし

### 顧客情報を編集する

- **目的**: 顧客の企業情報（社名・業種・規模・住所等）を更新する
- **前提**: 顧客が存在すること
- **権限**: 全ロール
- **結果**: 顧客情報が更新される
- **関連**: なし

### 顧客の担当者を追加する

- **目的**: 顧客企業の担当者情報を登録する
- **前提**: 顧客が存在すること
- **権限**: 全ロール
- **結果**: 担当者が追加される
- **関連**: なし

### 顧客の担当者を編集する

- **結果**: 担当者情報が更新される
