# invoice

### 請求を追加する

- **目的**: 契約に対する請求予定を登録する
- **前提**: 契約が有効であること
- **権限**: admin / manager
- **結果**: 請求が作成される
- **制約**: スポット契約の場合、請求合計が契約金額を超えないこと
- **関連**: contract

### 請求書を発行する

- **目的**: 予定の請求に対して請求書を発行済みにする
- **前提**: 請求が予定状態であること
- **権限**: admin / manager
- **結果**: 請求が請求済みになり、発行日が記録される
- **関連**: なし

### 入金を確認する

- **目的**: 請求に対する入金を記録する
- **前提**: 請求が請求済みであること
- **権限**: admin / manager
- **結果**: 請求が入金済みになり、入金日が記録される
- **関連**: なし

### 期日超過を記録する

- **目的**: 支払期日を過ぎた請求にマークする
- **前提**: 請求が請求済みであること
- **権限**: admin / manager
- **結果**: 請求が期日超過状態になる
- **関連**: なし
