# approval

### 承認リクエストを作成する

- **目的**: 承認が必要な申請を提出する
- **前提**: 承認テンプレートが存在すること
- **権限**: 全ロール
- **結果**: 承認リクエストが下書き状態で作成される
- **関連**: なし

### 承認リクエストを提出する

- **目的**: 下書きの承認リクエストを承認者に回す
- **前提**: 承認リクエストが下書き状態であること
- **権限**: 作成者
- **結果**: 承認リクエストが承認待ち状態になる
- **関連**: なし

### 承認する

- **目的**: 承認ステップを承認する
- **前提**: 自分のロールが承認ステップの承認者ロールと一致すること
- **権限**: 承認者ロール
- **結果**: ステップが承認される。全ステップ承認で承認リクエストが承認済みになる
- **関連**: なし

### 却下する

- **目的**: 承認リクエストを最終的に却下する
- **前提**: 承認リクエストが承認待ちであること
- **権限**: 承認者ロール
- **結果**: 承認リクエストが却下状態になる
- **関連**: なし

### 差し戻す

- **目的**: 承認リクエストの修正を申請者に求める
- **前提**: 承認リクエストが承認待ちであること
- **権限**: 承認者ロール
- **結果**: 承認リクエストが差し戻し状態になる
- **関連**: なし

### 再提出する

- **目的**: 差し戻された承認リクエストを修正して再提出する
- **前提**: 承認リクエストが差し戻し状態であること
- **権限**: 作成者
- **結果**: 承認リクエストが承認待ちに戻る
- **関連**: なし
