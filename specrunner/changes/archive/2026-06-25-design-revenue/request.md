## 背景

Claude Design（docs/design/Clearflow.dc.html の REVENUE DASHBOARD / SALES DETAIL / PLAN セクション）に合わせて売上画面のレイアウトを更新する。

## 現状コードの前提

- `src/app/(dashboard)/revenue/page.tsx` — 売上ダッシュボード。2 カラム（今月売上 + パイプライン予測）
- `src/app/(dashboard)/revenue/details/page.tsx` — 売上明細
- `src/app/(dashboard)/revenue/forecast/page.tsx` — 予実管理
- `docs/design/Clearflow.dc.html` — 売上ダッシュボード: 3 カラム KPI カード（今月売上, 確定見込み, パイプライン）+ 月次推移グラフ + 顧客別ランキング。売上明細: 期間フィルタ + 集計軸切替 + テーブル + CSV エクスポート。予実管理: 期間選択 + 目標設定 + プログレスバー

## 要件

1. **売上ダッシュボード — KPI カード**: 3 カラムグリッド（1fr 1fr 1fr）。今月の売上（大きめ緑文字）、確定見込み（契約 + 請求予定の金額）、パイプライン見込み（案件の想定金額合計）
2. **売上ダッシュボード — 月次推移**: グラフエリア（棒グラフのプレースホルダーまたは実装）。月クリックで明細に遷移
3. **売上ダッシュボード — 顧客別ランキング**: テーブルまたは横棒。顧客名クリックで顧客詳細に遷移
4. **売上明細 — フィルタバー**: 期間指定（開始日〜終了日）+ 集計軸切替（月別/顧客別/案件別）のタブ
5. **売上明細 — テーブル**: 集計軸に応じたカラム構成。行クリックで詳細に遷移。CSV エクスポートボタン
6. **予実管理 — 目標設定**: 期間選択（月次/四半期/年次）+ 目標金額のインライン編集
7. **予実管理 — 進捗表示**: 目標/実績/進捗率のプログレスバー + パイプライン見込みを加味した着地予測

## スコープ外

- グラフライブラリの導入（プレースホルダー or テーブル表示で代替可）
- ビジネスロジックの変更
